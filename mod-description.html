Connection Options

  • TCP (port 9876): Push notifications - receives game state automatically at turn end
  • HTTP (port 9877): REST API with 30+ endpoints for reading state and executing commands

Read Game State

  • Players: Nation, stockpiles, yield rates, legitimacy, techs, families, religion, laws
  • Cities: 80+ fields - yields, improvements, build queues, religion, culture, happiness
  • Characters: 85+ fields - traits, ratings, jobs, spouses, children, opinions, relationships
  • Units: Type, HP, promotions, cooldown, tile position, commander
  • Map/Tiles: Terrain, height, vegetation, resources, improvements, ownership
  • Diplomacy: Team relationships, war states, war scores, alliances
  • Tribes: Strength, cities, settlements, diplomacy with players

Execute Commands

166 commands for controlling the game:

  • Units: Move, attack, fortify, promote, disband, special actions
  • Cities: Build units/projects, hurry production, manage specialists
  • Research: Choose technologies
  • Economy: Buy/sell yields, trade luxuries, tribute
  • Diplomacy: Alliances, invasions
  • Turn: End turn, undo/redo, pause

Events

  • Characters: Births, deaths, marriages, leader changes, heir changes
  • Units: Created, killed
  • Cities: Founded, captured

Connection Options

  • TCP (port 9876): Persistent connection that receives push notifications on game events
  • HTTP (port 9877): REST API for on-demand queries (GET /state, /players, /cities, /characters, /character-events, /tribes)

Data Exposed

  • Players: Nation, stockpiles, yield rates, legitimacy, city/unit counts
  • Cities: Yields, improvements, build queues, religion, culture, happiness, governor, family
  • Characters: 75+ fields including traits, ratings, jobs, spouses, children, opinions, relationships
  • Character Events: Births, deaths, marriages, leader changes, heir changes
  • Diplomacy: Team relationships, war states, war scores, alliances
  • Tribes: Strength, cities, settlements, diplomacy with players
